2. What is an html attribute?

In HTML, an attribute is a property of an HTML element that provides additional information about the element. Attributes are included in the opening tag of an element and are typically used to define the characteristics or properties of that element. Examples of attributes include the "src" attribute in an "img" element, which specifies the source URL of an image, and the "href" attribute in an "a" element, which specifies the URL of a link

3. What is the purpose of the alt attribute in an img tag?

The alt attribute is used to give a different text in cases where the image or a page to be displayed cannot be found or rendered for any reason. it isrequired field in the image tag.

4. What is the difference between an absolute url and a relative url?

An absolute URL contains all the information needed to locate a resource on the internet, whereas a relative URL only contains the location of the resource relative to the current page.
